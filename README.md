# Lane Splitter
Created by Nathan Tisdale as a Web Programming project, for the Master of Arts in Graphic and Web Design program, at Minneapolis College of Art and Design
## Description
Written in JavaScript using the p5.js library.
## Game Play
You are a motorcyclist navigataing through traffic and must avoid cars by splitting lanes.
Rider can move left or right using arro keys on keyboard, or can tap/click to the left or right of the rider
Good luck
